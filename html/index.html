سه روش CSS

1•استفاده از سبک داخلی (Internal CSS):
در این روش، کدهای CSS به صورت مستقیم در بخش <style> درون تگ <head> صفحه HTML نوشته می‌شوند.

<style> body { color: red; } </style>

2•استفاده از سبک درون‌خطی (Inline CSS):
در این روش، استایل به صورت مستقیم داخل تگ HTML و با استفاده از ویژگی style تعریف می‌شود.

<p> style="color: red; font-size: 20px;"> متن </p>
3•استفاده از فایل خارجی (External CSS):
در این روش، فایل CSS در یک فایل جداگانه با پسوند .css ذخیره شده و با استفاده از تگ <html> در بخش <head> صفحه HTML به آن متصل می‌شود. <link rel="stylesheet" href="style.css">

dir و align

ویژگی dir برای تعیین جهت متن استفاده می‌شود.
این ویژگی مشخص می‌کند که متن از چپ به راست (LTR) یا راست به چپ (RTL) نمایش داده شود
مقادیر:
ltr: متن از چپ به راست.
rtl: متن از راست به چپ.

<p dir="rtl"> </p>

ویژگی align مشخص می‌کند که متن یا محتوای داخل یک عنصر HTML به چه صورت چپ‌چین، راست‌چین، وسط‌چین یا تراز شده باشد.
مقادیر:
left: محتوای عنصر به سمت چپ تراز می‌شود.
right: محتوای عنصر به سمت راست تراز می‌شود.
center: محتوای عنصر در وسط قرار می‌گیرد.
justify: متن به‌صورت تراز نمایش داده میشود.

<p align="right"> <p>
